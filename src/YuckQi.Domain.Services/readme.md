# YuckQi.Domain.Services

A .NET library for bootstrapping a domain services project. Intended as the foundation for service-layer logic that operates on domain entities and validation results.

## Dependencies

- [YuckQi.Domain](https://www.nuget.org/packages/YuckQi.Domain)
- [YuckQi.Domain.Validation](https://www.nuget.org/packages/YuckQi.Domain.Validation)

## Installation

```shell
dotnet add package YuckQi.Domain.Services
```
